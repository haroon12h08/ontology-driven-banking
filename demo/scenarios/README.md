# Demo Scenarios Configuration
This folder documents the rules, inputs, and target outcomes for the 6 demo scenarios.

- `home_loan`: Triggers DTI checks against `loan_rules.yaml`.
- `fraud_detection`: Velocity monitoring against `fraud_rules.yaml`.
- `investment`: Allocation optimization based on checking/savings ratios in `investment_rules.yaml`.
- `salary_credit`: Liquid wealth metrics update logic.
- `kyc`: Regulatory grace limitations inside `risk_rules.yaml`.
- `complaint`: Customer Operations resolution rules.
